Placez ici les fichiers de police Web (format recommandé : .woff2) pour héberger les polices localement.

Noms recommandés :
- PlayfairDisplay-Regular.woff2
- DMSans-Regular.woff2
- Caveat-Regular.woff2

Comment obtenir les fichiers depuis Google Fonts :
1. Allez sur https://fonts.google.com
2. Recherchez chaque police (Playfair Display, DM Sans, Caveat)
3. Sélectionnez les variantes nécessaires (Regular, 400/700 si besoin)
4. Téléchargez le pack via le bouton « Download family »
5. Dans l'archive téléchargée, convertissez/renommez les fichiers en .woff2 si nécessaire (Des outils comme fontforge ou des services en ligne peuvent aider)
6. Placez les fichiers .woff2 dans ce dossier `assets/fonts/` avec les noms recommandés ci-dessus.

Après avoir ajouté les fichiers, videz le cache du navigateur et rechargez la page. Les polices seront servies localement sans requêtes externes.
